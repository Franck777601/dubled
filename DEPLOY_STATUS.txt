🚀 DÉMARRAGE EN 3 ÉTAPES
═══════════════════════════════════════════════════════════════════════════

1. INSTALLER LES DÉPENDANCES
   $ npm install

2. LANCER LE SERVEUR
   $ npm start

3. ACCÉDER AUX 4 PAGES

   ISPA Français:      http://localhost:3000
   ISPA English:       http://localhost:3000/index-en.html
   ISBITAGES FR:       http://localhost:3000/ISBITAGES
   ISBITAGES EN:       http://localhost:3000/ISBITAGES/index-en.html
