# ISPA — Landing page

Remplacez le placeholder du numéro WhatsApp (`237XXXXXXXXX`) par le numéro réel.

Instructions rapides :

1. Ouvrez `[index.html](index.html)` dans un navigateur.
2. Pour remplacer le numéro globalement sans modifier les liens, ajoutez avant `script.js` un script comme :

```html
<script>
  window.ISPA_PHONE = '15559707710'; // exemple sans espaces
</script>
<script src="script.js"></script>
```

3. Remplissez la section `Filières principales` dans `index.html` avec la liste extraite du PDF.

Notes :
- Pas de prix affichés.
- Objectif unique : pousser le visiteur vers WhatsApp.
